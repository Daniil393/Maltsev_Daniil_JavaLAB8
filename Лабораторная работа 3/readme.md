  * Можно узнать текущее количество контактов.
  * Можно запросить в виде массива:
         o Все пары
         o Все телефоны
         o Все имена
  * Можно получить массив всех имен, указав часть названия (начиная с первого символа).

> Задание 5 – (Оружие)

Измените сущность Пистолет, полученную в задаче 1.5 3-ей лабораторной работы таким образом, чтобы она наследовалась 
от класса Weapon описанного ниже:

```java
public abstract class Weapon {

    private int ammo;

    public Weapon(int ammo) {
        if (ammo < 0) {
            throw new RuntimeException("Количество патронов не может быть отрицательным");
        }
        this.ammo = ammo;
    }

    public abstract void shoot();

    public int ammo() {
        return ammo;
    }

    public boolean getAmmo() {
        if (ammo == 0) {
            return false;
        }
        ammo--;
        return true;
    }

    public int load(int ammo) {
        if (ammo < 0) {
            throw new RuntimeException("Количество патронов не может быть отрицательным");
        }
        int tmp = this.ammo;
        this.ammo = ammo;
        return tmp;
    }
}
```

  * Имя, строка
  * Оружие, из задачи 4.1 3-ей лабораторной работы.
  * При создании объекта необходимо указать ему имя
  * Имя и оружие можно поменять и получить в любой момент без ограничения.
Основная способность Стрелка - умение стрелять. Если оружие есть, то выстрел происходит по 
правилам оружия, если его нет – то выводится текст “не могу участвовать в перестрелке”.
Создайте трех стрелков: одного без оружия, одного с пистолетом и одного с автоматом, и пусть 
каждый из них выстрелит.
___

В ходе выполнения данного задания был разработан класс Shooter и переделан класс AutomaticGun так чтобы он наследовался от класса Weapon, также использовался класс Pistol2.

### Класс Weapon
Класс состоит из поля, конструктора и методов.

*Поле*:
+  private int ammo - текущее количество патронов.

*Конструктор*:
+  public Weapon(int ammo) - создаёт оружие с заданным количеством патронов.
  1) Если передано отрицательное значение — выбрасывается RuntimeException;
  2) Иначе устанавливает начальное количество патронов.

*Методы*:
+  public abstract void shoot() - абстрактный метод выстрела.
Должен быть реализован в классах-наследниках.
+  public int ammo() - возвращает текущее количество патронов.
+  public boolean getAmmo() - пытается использовать один патрон:
  1) Если патронов нет (ammo == 0) — возвращает false.
  2) Если патроны есть — уменьшает их количество на 1 и возвращает true.
  3) Таким образом, метод одновременно проверяет наличие патрона и расходует его.
+  public int load(int ammo) - устанавливает новое количество патронов:
  1) Если передано отрицательное значение — выбрасывается RuntimeException.
  2) Сохраняет текущее количество патронов во временную переменную.
  3) Обновляет поле ammo.
  4) Возвращает предыдущее количество патронов.

### Класс Pistol2
Класс состоит из поля, конструктора и методов.

*Поле*:
+  private final int capacity - вместимость магазина.

*Конструктор*:
+  public Pistol2(int capacity)
  1) Вызывает конструктор родительского класса super(0), устанавливая начальное количество патронов в 0 (пистолет разряжен).
  2) Если вместимость отрицательная — выбрасывается IllegalArgumentException.
  3) Сохраняет значение вместимости в поле capacity.

*Методы*:
+  public void shoot() - выполняет один выстрел.
  1) Если есть патроны (getAmmo() возвращает true) — выводит "Бах!".
  2) Если патронов нет — выводит "Клац!".
+  public void shoot(int times) - выполняет несколько выстрелов подряд.
  1) Если times отрицательное — выбрасывается IllegalArgumentException.
  2) В цикле вызывает метод shoot() указанное количество раз.
+  public int reload(int bulletsToLoad) - перезаряжает пистолет.
  1) Если количество патронов отрицательное — выбрасывается IllegalArgumentException.
  2) Определяет текущее количество патронов (ammo()).
  3) Вычисляет свободное место в магазине.
  4) Если патроны помещаются полностью — загружает их и возвращает 0.
  5) Если не помещаются — заполняет магазин до максимума и возвращает количество лишних патронов.
+  public int unload() - разряжает пистолет.
  1) Сохраняет текущее количество патронов.
  2) Устанавливает количество патронов в 0 через load(0).
  3) Возвращает количество извлечённых патронов.
+  public boolean isLoaded() - проверяет, заряжен ли пистолет (есть ли хотя бы один патрон).
+  Геттер для поля capacity
+  Переопределенный toString() - формирует строковое представление

### Класс AutomaticGun2
Класс состоит из поля, конструктора и методов.

*Поле*:
+  private final int rateOfFire - скорострельность.

*Конструктор*:
+  public AutomaticGun2(int ammo, int rateOfFire) - создаёт автомат:
  1) Передаёт начальное количество патронов в конструктор родительского класса Weapon.
  2) Если скорострельность меньше или равна 0 — выбрасывается IllegalArgumentException.
  3) Иначе устанавливает значение rateOfFire.

*Методы*:
+  public void shoot() - выполняет автоматическую очередь выстрелов.
  1) Выполняет цикл rateOfFire раз.
  2) При каждом шаге вызывает getAmmo():
    + Если патрон есть — выводит "Бах!";
    + Если патронов нет — выводит "Клац!" и прекращает стрельбу (break).
+  public void shoot(int seconds) - стрельба в течение заданного количества секунд.
  1) Если seconds меньше или равно 0 — выбрасывается IllegalArgumentException.
  2) Общее количество попыток выстрела равно seconds * rateOfFire.
  3) Стрельба прекращается досрочно при отсутствии патронов.
+  Геттер для поля rateOfFire.
+  Переопределенный toString() - формирует строковое представление

> Задание 10 – (Возведение в степень)

Создайте метод принимающий две строки, в которых будут записаны числа X и Y. Возвращает 
метод результат возведения X в степень Y. Для преобразования строки в число следует 
использовать метод Integer.parseInt, а для возведения в степень метод Math.pow. Вызовите 
разработанный метод передав туда параметры командной строки полученные точкой входа в 
программу. Реализуйте метод так, что бы для возведения в степень и преобразования строки 
использовались короткие имена статических методов.
___

В ходе выполнения данного задания был разработан класс InputHelper и метод public static double power(String xStr, String yStr) в Main.

### Класс InputHelper
Класс состоит из одного метода.

*Метод*:
+  public static String readNumberString(Scanner sc, String prompt) - считывает строку с ввода и проверяет является ли она числом.
Параметры:
      Scanner sc — объект для чтения ввода.
      String prompt — текст к вводу.
Логика работы:
  1) Выводит сообщение-приглашение (prompt).
  2) Считывает строку и удаляет лишние пробелы (trim()).
  3) Пытается преобразовать строку в целое число через Integer.parseInt() (только для проверки).
  4) Если строка является числом — возвращает её.
  5) Если нет — выводит сообщение "Ошибка: введите целое число." и повторяет запрос.

### Метод power(String xStr, String yStr)

```java
public static double power(String xStr, String yStr) {
        int x = parseInt(xStr);
        int y = parseInt(yStr);

        if (x == 0 && y == 0) {
            throw new ArithmeticException("Выражение 0^0 не определено");
        }

        return pow(x, y);
    }
```

Параметры:
  + String xStr — строковое представление основания степени (X).
  + String yStr — строковое представление показателя степени (Y).

Логика работы:
  1) Преобразует строки в целые числа, используется метод Integer.parseInt().
  2) Проверяет особый случай:
      + if (x == 0 && y == 0)
     Если основание и показатель степени равны 0, выбрасывается ArithmeticException, так как выражение 0⁰ считается неопределённым.
  3) Если проверка пройдена, вычисляется степень, используется метод Math.pow().

Результат выполнения задания:

```java

Введите Y: -3
Результат: 0.125
```

```java

Введите Y: 0
Ошибка: Выражение 0^0 не определено
```
