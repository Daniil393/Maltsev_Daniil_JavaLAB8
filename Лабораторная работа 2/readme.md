Класс Main демонстрирует работу всех разработанных классов, выполняя требования заданий. Основная роль класса - создать объекты различных типов, вызвать их методы и визуально показать пользователю результаты работы.

Программа последовательно демонстрирует решения задач: создание людей, имён, сотрудников и отделов, а также функциональность пистолета.



> Задание 1.2 – (Создание сущности Человек)

В данном задании было необходимо создать сущность Person, которая задана данными параметрами:
Имя - строка, Рост - целое число

Может возвращать текстовое представление вида “Name, рост: height”, где Name и height это 
переменная с именем и ростом.

Необходимо создать и вывести на экран следующих людей:
+ Человек с именем “Клеопатра” и ростом 152
+ Человек с именем “Пушкин ” и ростом 167
+ Человек с именем “Владимир ” и ростом 189
___

В ходе выполнения данного задания был разработан класс Person.

### : Класс Person:
Класс состоит из полей, конструктора и методов.

*Поля*:
+  private String name - имя
+  private int height - рост

*Конструктор*:
+  public Person(String name, int height)

*Методы*:
+  Геттеры для всех полей
+  Переопределенный toString() - формирует строковое представление

Результат выполнения задания:

```java
Клеопатра, рост: 152
Пушкин, рост: 167
Владимир, рост: 189
```

> Задание 1.3 – (Создание сущности Имя)

В данном задании было необходимо создать сущность Имя, которая описывается тремя параметрами: Фамилия, Личное имя, 
Отчество. Имя может быть приведено к строковому виду, включающему традиционное 
представление всех трех параметров: Фамилия Имя Отчество (например “Иванов Иван 
Иванович”). Необходимо предусмотреть возможность того, что какой-либо из параметров может 
быть не задан, и в этом случае он не учитывается при приведении к текстовому виду.

Необходимо создать и вывести на экран следующие имена:
+ Клеопатра
+ Пушкин Александр Сергеевич
+ Маяковский Владимир


Обратите внимание, что при выводе на экран, не заданные параметры никак не участвуют в 
образовании строки.
___

В ходе выполнения данного задания был разработан класс Name.

### : Класс Name:
Класс состоит из полей, конструкторов и метода.

*Поля*:
+  private String surname - фамилия
+  private String firstname - имя
+  private String patronymic - отчество

*Конструкторы*:
+  public Name(String firstname)
+  public Name(String surname, String firstname)
+  public Name(String surname, String firstname, String patronymic)

*Метод*:
+  Переопределенный toString() - формирует строковое представление

Результат выполнения задания:

```java
Клеопатра
Пушкин Александр Сергеевич
Маяковский Владимир
```
