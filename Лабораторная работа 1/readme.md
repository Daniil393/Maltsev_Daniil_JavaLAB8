Далее в методе реализуется конструкция switch-case, которая позволяет наиболее удобным способом обратиться к методам класса lab1.
Эта конструкция позволяет обрабатывать различные случаи (cases) в зависимости от значения переменной. Каждый case конструкции switch-case
содержит дружественный интерфейс для ввода данных и вызов определенного метода класса public class lab1.

Также каждый case содержит вызов вспомогательных методов класса lab1, для проверки ввода целочисленного значения или одного символа.

Рассмотрим реализацию каждого метода этого класса:

1. public int sumLastNums(int x) - вычисление суммы двух последних цифр числа x.
Метод получает одно целое число, далее из него выделяются последняя цифра через операцию x % 10 и предпоследняя через (x / 10) % 10. Обе цифры берутся по модулю, после чего возвращается их сумма.

2. public boolean isPositive (int x) - проверка числа на положительность.
На вход передаётся одно число x. Через условие if определяется, больше или меньше число нуля. При x = 0 выводится пояснение, что ноль не относится ни к положительным, ни к отрицательным значениям.

3. public boolean isUpperCase(char x) - определение регистра английской буквы.
Метод получает символ x, далее через сравнение с диапазонами 'A'..'Z' и 'a'..'z' определяется, является ли он заглавным или строчным. В зависимости от результата метод выводит соответствующее сообщение и возвращает булево значение.

4. public boolean isDivisor(int a, int b) - проверка делимости двух чисел.
Метод принимает два числа. Если оба равны нулю, то выводится сообщение об ошибке. Далее проверяется, делится ли одно число на другое без остатка. В зависимости от результата выводится информация о делимости и возвращается true или false. Если же не делятся то выводится сообщение ("Ни одно из чисел не делится на другое нацело") и возвращается false.

5. public int LastNumSum(int a, int b) - суммирование последних цифр двух чисел.
Метод выделяет последние цифры чисел a и b с помощью операции % 10 и возвращает сумму их модулей.

6. public double safeDiv(int x, int y) - безопасное деление двух чисел.
При делении x на y метод предварительно проверяет делитель. Если y = 0, возвращается 0. В остальных случаях результат приводится к типу double. Если деление получилось целым - возвращается целое значение.

7. public String makeDecision(int x, int y) - формирование строки сравнения двух чисел.
Метод сравнивает x и y с помощью конструкции if-else и возвращает строку вида "x > y", "x < y" или "x == y".

8. public boolean sum3(int x, int y, int z) - проверка, можно ли получить одно число суммой двух других.
Метод проверяет все комбинации x + y, x + z и y + z. При совпадении суммы с третьим числом выводится сообщение и возвращается true, иначе - false.

9. public String age(int x) - формирование корректной формы слова «год».
Метод анализирует последнюю цифру числа x и диапазон исключений 11 - 14. В зависимости от правил русского языка возвращает строку вида "x год", "x года" или "x лет".

10. public void printDays(String x) - вывод дней недели, начиная с указанного дня.
Метод переводит строку x в нижний регистр и использует конструкцию switch-case без операторов break (кроме последнего case) для последовательного вывода всех последующих дней до воскресенья.

11. public String reverseListNums(int x) - формирование строки чисел от x до 0.
Метод определяет направление цикла в зависимости от знака x: уменьшается до 0 при положительном числе или увеличивается до 0 при отрицательном. Все числа собираются в строку с пробелами.

12. public int pow(int x, int y) - возведение числа x в степень y.
Метод выполняет y последовательных умножений через цикл for и возвращает результат.

13. public boolean equalNum(int x) - проверка, что все цифры числа одинаковые.
Метод выделяет крайнюю цифру числа x, затем последовательным делением сравнивает каждую следующую цифру. При обнаружении отличия выводится сообщение и возвращается false и выводит соответствующее сообщение. При полном совпадении всех цифр возвращается true и выводит нужное сообщение.

14. public void leftTriangle(int x) - вывод левого прямоугольного треугольника из символов '*'.
Метод печатает треугольник высотой x. Каждый следующий ряд содержит на один символ больше предыдущего.

15. public void guessGame() - реализация игры «Угадай число».
Генерируется случайное число в диапазоне 0 - 9. Пользователь вводит числа с клавиатуры, пока не угадает скрытое значение. Метод учитывает ввод ошибок и количество попыток. --------------------------------------------------------------------------------------------

16. public int findLast(int[] arr, int x) - поиск последнего вхождения числа в массиве.
Метод запускает цикл от конца массива к началу, и при совпадении значения элемента с x возвращает индекс. Если значение не найдено - возвращает -1.

17. public int[]add (int[] arr, int x, int pos) - вставка элемента в массив по указанной позиции.
Метод проверяет корректность позиции вставки. Затем создаёт новый массив на один элемент длиннее исходного, копирует в него элементы до позиции pos, вставляет x и затем добавляет оставшиеся элементы.

18. public void reverse(int[] arr) - реверс массива.
Метод меняет элементы массива местами, используя два индекса - левый и правый. Процесс продолжается до их встречи.

19. public int[] concat(int[] arr1, int[] arr2) - объединение двух массивов в один.
Метод создаёт новый массив длиной arr1 + arr2. Первая часть заполняется элементами arr1, вторая - элементами arr2.

20. public int[] deleteNegative(int[] arr) - удаление всех отрицательных чисел массива.
Метод подсчитывает количество неотрицательных элементов, создаёт новый массив нужной длины и копирует в него только элементы ≥ 0. Затем возвращается новый массив.



Вспомогательные методы класса lab1:
1. public int ReadInt(Scanner sc, String prompt) - метод для ввода целочисленного значения.
Метод выводит сообщение и ожидает целое число. При ошибке выводит соответствующее сообщение и запрашивает ввод повторно.

2. public char ReadChar(Scanner sc, String prompt) - метод для ввода одного символа.
Метод выводит сообщение, после чего читает строку. Если длина строки равна 1 - возвращает символ, иначе выводит сообщение об ошибке и запрашивает ввод повторно.
